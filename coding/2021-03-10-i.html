# 日期时间

  • date-fns
    • 提供了最全面，最简单且一致的工具集，用于在浏览器和 Node.js 中操作 JavaScript 日期。
    • 官网
    • GitHub
    • npm
  • Moment.js
    • 一个 JavaScript 日期库，用于解析，验证，操作和格式化日期。
    • 官网
    • GitHub
    • npm
  • Day.js
    • 和 Moment.js 具有相同 API，是 Moment.js 的轻量级替代品，仅 2kB。
    • 官网
    • GitHub
    • npm

  • axios

    • 基于 Promise 的 HTTP 客户端，用于浏览器和 node.js

    • npm

  • js-cookie

    • 一个简单，轻巧的 JavaScript API，用于处理 Cookie
    • GitHub
    • npm

# 图像处理

  • DOM to Image

    • GitHub
    • npm

# 数据可视化

  • Apache ECharts

    • 一个免费的，功能强大的图表和可视化库（不争气的百度卖给 Apache 了）
    • 官网
    • GitHub
    • npm

  • AntV

    • 蚂蚁集团数据可视化团队开发的数据可视化解决方案。包括 G2|G2Plot、F2|F2Native、G6|Graphin、X6、L7、AVA。
    • 官网

# 拖拽

  • dnd.js
    • 不依赖任何第三方库的拖放库，兼容低版本浏览器，兼容移动端，自带常用动画效果。
    • npm
  • Droppable
    • 一个轻量级的，响应式的现代 JavaScript 拖放库。
    • 官网
    • GitHub
    • npm

# 编辑器

  • TinyMCE
    • 功能强大、所见即所得的富文本编辑器
    • 中文文档
    • 官网
    • GitHub
    • npm
  • wangEditor
    • Typescript 开发的 Web 富文本编辑器， 轻量、简洁、易用、开源免费
    • 官网
    • GitHub
    • npm
  • JSON Editor
    • 基于 Web 的工具，用于查看、编辑、格式化和验证 JSON。它具有多种模式，例如树编辑器、代码编辑器和纯文本编辑器。
    • GitHub
    • npm
  • CodeMirror
    • JavaScript 实现的通用文本编辑器。它专门用于编辑代码，并具有 100 多种语言模式和各种插件，可实现更高级的编辑功能。每种语言都带有功能齐全的代码和语法突出显示，以帮助阅读和编辑复杂的代码。
    • 官网
    • GitHub
    • npm
